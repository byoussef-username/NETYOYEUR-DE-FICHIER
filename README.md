# Nettoyeur de fichiers vides

Interface graphique Tkinter permettant de supprimer les fichiers vides et les dossiers vides d'un répertoire, de façon récursive.

## Fonctionnalités

- Sélection d'un dossier via une fenêtre de dialogue
- Suppression des fichiers de taille nulle
- Suppression des dossiers devenus vides après nettoyage
- Journal des suppressions affiché dans l'interface

## Prérequis

- Python 3

## Utilisation

```
python netoyeur_de_fichier_vide.py
```
